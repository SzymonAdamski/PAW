## Konfiguracja `.env`

```env
VITE_GOOGLE_CLIENT_ID=ID_KLIENTA_GOOGLE
VITE_SUPER_ADMIN_EMAIL=EMAIL_ADMINA
VITE_DATA_STORAGE=localStorage
```

Dla bazy Firestore:

```env
VITE_DATA_STORAGE=firestore
VITE_FIREBASE_API_KEY=KLUCZ_API
VITE_FIREBASE_AUTH_DOMAIN=DOMENA_AUTH
VITE_FIREBASE_PROJECT_ID=ID_PROJEKTU
VITE_FIREBASE_APP_ID=ID_APLIKACJI
```
